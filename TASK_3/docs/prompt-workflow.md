# Prompt Workflow Documentation

## Workflow Overview

This prompt system follows the same sequence used by SEO teams when planning content for business websites.

## Step 1: Build The Client Brief

Use `prompts/01-client-brief-prompt.md` to define:

- Business niche
- Location
- Services
- Audience
- Business goal
- Tone

For this project:

- Niche: Skin care clinic
- City: Visakhapatnam
- Goal: Consultation leads and authority
- Tone: Premium, helpful, dermatology-aware

## Step 2: Generate Keyword Clusters

Use `prompts/02-keyword-cluster-prompt.md` to create:

- One broad pillar topic
- Five supporting topics
- Keyword intent
- Funnel stage
- Internal links

## Step 3: Create SEO Outlines

Use `prompts/03-blog-outline-prompt.md` for every blog before drafting. This prevents random content and ensures each blog has:

- One clear H1
- Structured H2 and H3 sections
- FAQ opportunities
- Local SEO hooks
- Conversion-focused ending

## Step 4: Write Long-Form Content

Use `prompts/04-long-form-blog-prompt.md` to turn each outline into helpful content. The output should educate the reader while guiding them toward a consultation.

## Step 5: Optimize For Local SEO

Use `prompts/05-local-seo-optimization-prompt.md` to improve city relevance, trust signals, and internal linking.

## Internal Linking Model

The pillar blog acts as the main authority page:

- Supporting blogs link to the pillar page.
- The pillar page links to all supporting blogs.
- Service-specific blogs link to relevant treatment pages.
- FAQs link to consultation pages where appropriate.

## Reuse Instructions

To reuse this system for another client:

1. Replace the niche.
2. Replace the city.
3. Replace the services.
4. Generate a new keyword cluster.
5. Create outlines.
6. Draft blogs.
7. Run the local SEO optimizer.
