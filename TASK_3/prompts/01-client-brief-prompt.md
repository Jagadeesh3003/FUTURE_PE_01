# Prompt 1: Client Brief Builder

## Purpose

Use this prompt to collect or generate a clear SEO brief before writing any blog content.

## Prompt

You are an SEO content strategist for business websites.

Create a client SEO brief using the details below:

- Business niche: [BUSINESS_NICHE]
- City/location: [CITY]
- Main services: [SERVICES]
- Ideal customer: [CUSTOMER_PROFILE]
- Business goal: [LEADS / TRUST / BOOKINGS / EDUCATION / AUTHORITY]
- Tone: [PROFESSIONAL / FRIENDLY / PREMIUM / CLINICAL]

Return the brief in this structure:

1. Business overview
2. Target audience
3. Primary SEO opportunity
4. Local SEO angle
5. Trust signals to include
6. Conversion goal
7. Content topics the website should own

Make the brief practical for an SEO agency and a business owner.
