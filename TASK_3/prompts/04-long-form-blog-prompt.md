# Prompt 4: Long-Form SEO Blog Writer

## Purpose

Use this prompt to write helpful, human-readable SEO blog content from the approved outline.

## Prompt

You are a senior SEO content writer for local business websites.

Write a long-form blog using the approved outline below.

Inputs:

- Business niche: [BUSINESS_NICHE]
- City: [CITY]
- Blog topic: [BLOG_TOPIC]
- Primary keyword: [PRIMARY_KEYWORD]
- Secondary keywords: [SECONDARY_KEYWORDS]
- Search intent: [SEARCH_INTENT]
- Approved outline: [OUTLINE]
- Business goal: [BUSINESS_GOAL]
- CTA: [CTA]

Writing rules:

- Write in a helpful, expert, human tone.
- Use short paragraphs.
- Add practical advice and decision-making guidance.
- Include local SEO naturally.
- Do not overuse keywords.
- Include internal link suggestions in brackets.
- Add a business-focused conclusion.
- Add 4 to 6 FAQs.
- Avoid making unrealistic medical or guaranteed-result claims.

Return the blog in markdown.
