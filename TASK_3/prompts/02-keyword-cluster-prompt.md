# Prompt 2: Keyword And Content Cluster Generator

## Purpose

Use this prompt to generate a pillar blog topic, supporting blog topics, keyword groups, and search intent.

## Prompt

You are an expert SEO strategist.

Create a content cluster for a business website.

Inputs:

- Business niche: [BUSINESS_NICHE]
- City: [CITY]
- Main service category: [MAIN_TOPIC]
- Services offered: [SERVICES]
- Audience: [AUDIENCE]

Generate:

1. One pillar blog topic
2. Five supporting blog topics
3. Primary keyword for each blog
4. Secondary keywords
5. Search intent for each keyword
6. Funnel stage: awareness, consideration, or decision
7. Internal linking plan
8. Recommended CTA for each blog

Rules:

- Include city and service keywords naturally.
- Avoid keyword stuffing.
- Prioritize helpful content that can earn trust.
- Make the cluster suitable for a local business website.
