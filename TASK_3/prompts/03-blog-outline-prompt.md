# Prompt 3: SEO Blog Outline Generator

## Purpose

Use this prompt to generate SEO-friendly H1, H2, and H3 headings before writing the blog.

## Prompt

You are an SEO blog editor.

Create a detailed SEO blog outline for the topic below:

- Blog topic: [BLOG_TOPIC]
- Primary keyword: [PRIMARY_KEYWORD]
- Secondary keywords: [SECONDARY_KEYWORDS]
- City: [CITY]
- Business type: [BUSINESS_TYPE]
- Search intent: [SEARCH_INTENT]

Return:

1. SEO title
2. Meta description under 160 characters
3. URL slug
4. H1
5. H2 and H3 structure
6. Internal link suggestions
7. External authority link suggestions
8. FAQ questions
9. Business-focused conclusion angle

Rules:

- Use one clear H1.
- Make headings readable and natural.
- Include local SEO terms only where they fit.
- Match the user's search intent.
