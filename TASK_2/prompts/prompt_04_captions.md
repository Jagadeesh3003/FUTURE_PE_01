# Prompt 04 — Social Media Captions

## 🎯 Purpose
Generate ready-to-post captions for each platform to accompany the UGC video ads.

---

## ✅ Filled Prompt Used

```
You are a social media copywriter specialising in local D2C brands.

Write social media captions for the following brand and content:

Brand: Threads & Thread Co., Hyderabad
Content Type: UGC-style clothing ad
Platforms: Instagram, YouTube Shorts, Facebook
Tone: Casual, local, relatable — not salesy

For each platform write:
- A caption (2–4 lines max)
- Relevant hashtags (5–8 for Instagram, 3–4 for YouTube, 2–3 for Facebook)
- One engagement hook (question or "tag a friend" line)

Rules:
- Instagram: Short, punchy, emoji-accented
- YouTube: Slightly descriptive, keyword-friendly
- Facebook: Warm, community-driven
- Never use "DM us to order" as the only CTA — vary it
```

# Prompt 05 — CTA Bank

## 🎯 Purpose
Build a reusable bank of 10 CTAs in different styles so every ad ends differently and doesn't feel repetitive.

---

## ✅ Filled Prompt Used

```
You are a conversion copywriter for social media ads.

Generate a CTA bank of 10 different calls-to-action for the following brand:

Brand: Threads & Thread Co., Hyderabad
Goal: Drive Instagram follows, website visits, and store walk-ins
Tone: Friendly, low-friction, conversational

CTA Types to Include:
1. Soft visit CTA (non-pushy)
2. Curiosity CTA (makes them want to explore)
3. Community CTA (tag a friend)
4. Urgency CTA (limited stock / season)
5. Social follow CTA
6. Location CTA (Hyderabad-specific)
7. Discount/offer CTA
8. Story CTA (swipe up / link in bio)
9. Question CTA (comment-bait)
10. FOMO CTA

Rules:
- Each CTA must be under 20 words
- Sound like a person suggesting something, not a brand demanding action
- At least 3 must reference Hyderabad or local culture
```
