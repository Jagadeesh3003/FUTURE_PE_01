# Prompt 02 — Full UGC Ad Scripts

## 🎯 Purpose
Generate 3 complete UGC-style ad scripts following the Problem → Solution → Proof → CTA framework.

---

## 📋 Prompt Template

```
You are a UGC scriptwriter for a content marketing agency.

Write 3 different UGC-style ad scripts for the following brand:

Brand Name: [BRAND NAME]
Product: [PRODUCT/SERVICE]
Target Audience: [AUDIENCE]
Ad Length: 30–45 seconds (spoken word, on-camera)
Tone: Casual, conversational, authentic — like a real customer review, NOT a brand ad

For each script, follow this structure:
- HOOK (0–3 sec): Scroll-stopping opening line
- PROBLEM (3–10 sec): Relate to a pain point the audience has
- SOLUTION (10–25 sec): Naturally introduce the brand/product as the fix
- PROOF (25–35 sec): One social proof signal (reviews, personal result, crowd reaction)
- CTA (35–45 sec): One low-friction action

Rules:
- Never say "I was paid to say this" or sound scripted
- Avoid superlatives: "best", "amazing", "incredible"
- Use natural filler where appropriate ("honestly", "like", "okay so")
- Each script must have a different hook and different angle
- End CTA must feel like a friendly suggestion, not a sales push
```

---

## ✅ Filled Prompt Used

```
You are a UGC scriptwriter for a content marketing agency.

Write 3 different UGC-style ad scripts for the following brand:

Brand Name: Threads & Thread Co.
Product: Affordable fusion clothing — Indo-western, streetwear, and occasion wear
Location: Hyderabad
Target Audience: College students and young professionals aged 18–30 in Hyderabad
Ad Length: 30–45 seconds (spoken word, on-camera style)
Tone: Casual, like a Hyderabad college student or young professional talking to their Instagram followers

For each script, follow this structure:
- HOOK (0–3 sec): Scroll-stopping opening line
- PROBLEM (3–10 sec): Relate to a pain point the audience has
- SOLUTION (10–25 sec): Naturally introduce the brand/product as the fix
- PROOF (25–35 sec): One social proof signal
- CTA (35–45 sec): One low-friction friendly suggestion

Rules:
- Never sound like a brand talking — always sound like a real customer
- Avoid: "best brand", "amazing quality", "you won't regret it"
- Use natural filler where appropriate ("okay so", "honestly", "like")
- Script 1: Pain point angle (budget + style)
- Script 2: Occasion/event angle (function, wedding, college fest)
- Script 3: Identity angle (dressing for who you're becoming)
- Add Hyderabad local flavour (references to areas, culture, occasions)
```

---

## 🔁 How to Reuse
- Change the "Script angle" instructions for different campaign goals
- For product launches: add "Script 1 = unboxing angle"
- For seasonal campaigns: add "Script 1 = festive/Diwali angle"
