# Prompt 03 — Platform-Specific Adaptation

## 🎯 Purpose
Adapt the same core UGC script for Instagram Reels, YouTube Shorts, and Facebook Reels — each platform has a different audience energy and format expectation.

---

## 📋 Prompt Template

```
You are a social media content strategist.

Take the following UGC ad script and adapt it for 3 platforms:

[PASTE BASE SCRIPT HERE]

Platform Adaptation Rules:

INSTAGRAM REELS:
- Length: 15–30 seconds
- Energy: High, fast-paced, punchy
- Visual cue: Add [on-screen text] suggestions
- Caption style: Short, emoji-forward, 3–5 hashtags
- Audience: Trend-aware, visually driven

YOUTUBE SHORTS:
- Length: 30–45 seconds
- Energy: Slightly more story-driven, slightly slower pace
- Add: A brief "okay so here's what happened" storytelling moment
- CTA: "Link in description" or "Search [brand name]"
- Audience: Discovery-focused, open to slightly longer hooks

FACEBOOK REELS:
- Length: 30–60 seconds
- Energy: Community-feel, warm, relatable
- Add: One social proof line (ratings, others' reactions)
- CTA: "Tag a friend who needs this"
- Audience: Slightly older (24–35), value-driven, trust-first

Output Format:
For each platform: adapted script + caption + suggested on-screen text overlays
```
