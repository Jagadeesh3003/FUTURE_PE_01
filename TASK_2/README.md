## 🧠 Prompt Logic

This task uses a **4-Layer UGC Ad Prompt Framework** built on proven ad psychology:

### The Core Formula
```
HOOK (stop the scroll in 3 sec)
→ PROBLEM (make them feel seen)
→ SOLUTION (introduce the brand naturally)
→ PROOF (social trust / result)
→ CTA (one clear action)
```

### Prompt Design Principles
1. **UGC Voice First** — Write like a real customer talking to a friend, not a brand talking to a customer
2. **Hook Variety** — Generate 5 different hooks per product (curiosity, pain point, social proof, contrast, bold claim)
3. **Platform Calibration** — Adjust length & energy per platform (Instagram = punchy, YouTube = slightly longer, Facebook = story-driven)
4. **No Salesy Language** — Avoid "Buy now!", "Best brand ever!", "Amazing quality!" — keep it conversational
5. **Conversion Intent** — Every script ends with one clear, low-friction CTA

## ✅ Key Learnings
- UGC ads that start with a relatable problem outperform product-first openers
- Hook variation testing is the #1 lever for improving ad performance
- Platform tone matters: Instagram = casual & fast, YouTube Shorts = slightly story-driven, Facebook = community-feel
- Prompts that specify "do NOT sound like an ad" produce the most authentic UGC copy

## 🔁 Reusability
This prompt framework works for any brand:
- Skincare D2C → change pain point to "breakouts / dull skin"
- Gym / Fitness → change to "I used to skip the gym until..."
- SaaS tool → change to "I was wasting 3 hours a day on this until..."
- Food brand → change to sensory-driven hooks
