## 🧠 Prompt Logic

This task uses a **structured prompt framework** with 4 reusable prompt templates:

### Prompt Design Principles
1. **Role Assignment** — Tell the AI to act as a professional copywriter
2. **Business Context** — Provide specific business name, type, city, audience
3. **Tone Instruction** — Specify tone (friendly, warm, inviting for a café)
4. **Output Format** — Request specific sections (headline, sub-headline, CTA, etc.)
5. **Constraint** — "No generic AI text — make it feel human and local"

## ✅ Key Learnings
- Structured prompts produce far better copy than vague instructions
- Adding business-specific details (location, audience, USP) removes generic AI tone
- Role + Context + Format + Constraint = high-quality output formula
- The same prompt framework is reusable for any local business type
