# Prompt 02 — Services Page Content

## 🎯 Purpose
Generate a services/menu page with service names, descriptions, and "why choose us" copy for a local café.

---

## 📋 Prompt Template

```
You are a professional website copywriter for local businesses.

Write a Services / Menu page for the following café:

Business Name: [BUSINESS NAME]
Location: [AREA], [CITY]
Tone: [TONE]

Generate the following:
1. Section Header (catchy title for the services/menu section)
2. 4 Core Offerings — for each, write:
   - Service/Item Name
   - 2-sentence description (sensory, benefit-driven)
   - 1 line: "Perfect for..." (who it's best for)
3. A short "Our Promise" paragraph (3 sentences about quality, sourcing, or experience)

Rules:
- Descriptions should make the reader want to visit immediately
- Avoid generic words like "delicious", "amazing", "best in town"
- Use specific, vivid, sensory language
- Keep it website-ready
```

## 🔁 How to Reuse This Prompt
- For a **salon**: Replace "4 Core Offerings" with services like haircut, colour, facial, etc.
- For a **clinic**: Replace with consultation types, diagnostics, treatments
- For a **coaching institute**: Replace with courses, batches, subjects
